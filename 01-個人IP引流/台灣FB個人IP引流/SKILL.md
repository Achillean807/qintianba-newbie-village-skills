# 台灣 FB 個人 IP 引流

你不是在寫一篇看起來很會行銷的貼文。
你是在幫一個有真實人生、有真實工作、有真實判斷的人，把內容寫到讓讀者願意停下來、願意看完、願意私訊。

## 這個技能什麼時候用

遇到以下需求就開：
- 個人品牌貼文
- 觀點文
- 日常故事文
- 工作紀錄文
- 招生活動前的暖身文
- 私訊前置導流文
- 社團貼文

## 平台認知

台灣 FB 到現在還是很吃這幾件事：
- 讀者年齡偏成熟
- 長文仍然有機會贏短文
- 先像人，再像品牌
- 越硬賣越容易被滑掉
- 真實故事、反差觀點、具體場景，停留率通常比空泛金句高

## 內容原則

### 1. 先抓場景，不先喊口號
不要一開頭就說「今天想跟大家分享」。
直接把讀者丟進畫面裡。

### 2. 先給判斷，再給推銷
先讓讀者感覺你真的看過、做過、踩過坑。
沒有判斷的內容，很難引發信任。

### 3. 貼文不是結案報告
每段只做一件事。
一段講場景。
一段講轉折。
一段講你怎麼看。
最後才收 CTA。

### 4. CTA 要像順手推門，不要像拿棍子趕人
CTA 可以是：
- 留言領資料
- 私訊關鍵字
- 來聽說明會
- 想看完整版本再敲我

但不要整篇都像銷售頁。

## 預設結構

```
1. 開頭 Hook：一句反差、痛點、現場畫面
2. 情境展開：發生了什麼
3. 觀點轉折：你看到的關鍵不是表面那一層
4. 可帶走的重點：3 個內
5. 結尾 CTA：只留一個動作
```

## 五種常用貼文類型

### 類型 1：觀點型
適合建立人設與判斷力。
開頭要有明確立場。

### 類型 2：故事型
用一個人、一個對話、一個現場，把讀者帶進來。
適合做信任建立。

### 類型 3：教學型
不是把知識倒出來。
是先講為什麼這件事卡住，再給最短可做版本。

### 類型 4：拆解型
適合針對市場迷思、常見誤區、錯誤期待做拆解。

### 類型 5：活動暖身型
不直接報名轟炸。
先用故事、案例、痛點，把人帶到願意看活動。

## 寫作規則

- 優先用繁體中文
- 句子偏短
- 手機閱讀要順
- 少用空泛形容詞
- 不要端出一堆術語嚇人
- 若是村長系任務，先搭配 `social-writer` 再落地

## 結尾 CTA 模板

- 如果你也卡在這一段，留言「想看」我整理完整版
- 如果你想把這件事做成自己的版本，私訊我
- 這題我之後會在說明會拆開講，有興趣我再把資訊放上來
- 你如果也遇過同樣問題，留言讓我知道

## 輸出格式

### 格式 A：單篇貼文
- 主文
- 一句封面字
- 一句留言引導

### 格式 B：一週內容主題
- 7 則主題方向
- 每則一句 Hook
- 對應 CTA

### 格式 C：活動前暖身三連發
- 第一篇：痛點與誤解
- 第二篇：案例與反轉
- 第三篇：活動導流

## 自我檢查

- 讀起來像真人，不像模板嗎
- 有具體畫面，不是只有抽象概念嗎
- 有沒有明確立場或洞見
- CTA 是否只有一個
- 看完會不會想私訊，而不是只覺得你很吵
