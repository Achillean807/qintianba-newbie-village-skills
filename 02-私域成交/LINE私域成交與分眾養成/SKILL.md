# LINE 私域成交與分眾養成

LINE 不是拿來亂炸訊息的。
它是台灣市場裡最接近收件匣、最接近私聊、也最容易因為亂推而被封鎖的地方。

## 什麼時候用

- 要把 FB / IG / 活動流量接進 LINE
- 要做 LINE 官方帳號養名單
- 要設計 Rich Menu
- 要設計關鍵字自動回覆
- 要規劃新好友歡迎流程
- 要分眾推播而不是對全部人亂發

## 先分清楚三種場景

### 1. LINE 官方帳號
適合品牌、工作室、課程、活動、服務銷售。

### 2. LINE 社群
適合陪跑、互動、同學會、主題社群。

### 3. LINE 群組
適合小型協作、短期活動、內部溝通。

## 核心觀念

### 1. 一則訊息只做一件事
一則訊息如果同時要你看影片、點報名、填問卷、加好友、買產品，通常每一件都做不好。

### 2. 名單要分層，不要混成一鍋
最基本至少分：
- 來源
- 興趣
- 階段
- 是否已成交

### 3. 用免費互動取代昂貴群發
可以用：
- Rich Menu
- 關鍵字回覆
- 歡迎流程
- 分眾標籤

這些做好，比一直群發省很多。

## 建議分眾方式

### 來源標籤
- FB 貼文
- 活動報名
- 現場掃碼
- 客戶介紹

### 興趣標籤
- 想學
- 想買服務
- 想參加活動
- 只是先看看

### 階段標籤
- 新加入
- 已互動
- 已報名
- 已成交
- 沉睡

## 新好友歡迎流程

```
1. 歡迎訊息：先交代你是誰、這裡能拿到什麼
2. 引導選擇：讓對方自己選關心的主題
3. 打標籤：根據回覆做分眾
4. 給第一份價值：清單、懶人包、回放、教學
5. 再安排下一步 CTA
```

## Rich Menu 規劃原則

最常被點的位置放這些：
- 主打服務
- 最新活動
- 免費資源
- 預約 / 詢問
- 品牌介紹
- 常見問題

不要把 Rich Menu 做成資訊墓園。
每一格都要有明確去處。

## 訊息寫法

### 好訊息長這樣
- 開頭一句就有價值
- 中間只講一件事
- 結尾只留一個 CTA

### 爛訊息長這樣
- 先寒暄三句
- 再講自己多努力
- 最後丟五個連結

## 常用訊息場景

### 場景 1：活動報名前提醒
目標：提醒，不製造壓力。

### 場景 2：活動後回放與追擊
目標：把看過的人往下一步推。

### 場景 3：沉睡名單喚醒
目標：先用高價值內容，不要直接賣。

### 場景 4：成交前最後一推
目標：回答疑慮，而不是催命。

## 輸出格式

### 格式 A：一則 LINE 群發
- 標題一句
- 內文 100-180 字
- CTA 一個
- 圖片需求建議

### 格式 B：Rich Menu 規劃
- 6 格內容
- 每格點擊動作
- 文案與視覺建議

### 格式 C：新好友 3 則歡迎流程
- 第 1 則：定位與承諾
- 第 2 則：分眾互動
- 第 3 則：第一個 CTA

### 格式 D：分眾策略表
- 標籤邏輯
- 觸發條件
- 適合發什麼

## 禁忌

- 不要深夜亂發
- 不要連續幾天硬賣
- 不要每則都像促銷傳單
- 不要沒有標籤就發全部人
- 不要用看不懂的系統話術嚇死一般人
