# 說明會與工作坊轉單

活動不是把一堆資訊搬上台。
活動的本質是把觀望的人，往更高信任的下一步推進。

## 使用情境

- 體驗會
- 說明會
- 分享會
- 小型工作坊
- 招生直播
- 私密場說明
- 活動頁與報名文案

## 先釐清四件事

### 1. 這場活動的唯一目的
只能選一個主目標：
- 收報名
- 收私訊
- 收預約
- 收訂金
- 收名單

### 2. 聽的人現在卡在哪
- 不懂這件事值不值得
- 不確定自己適不適合
- 覺得太難
- 覺得太貴
- 覺得之後再說

### 3. 活動後要接去哪
- 表單
- LINE
- 一對一訪談
- 付款頁

### 4. 這場活動賣的是什麼改變
不要只賣流程。
要講清楚來了之後，對方會少走什麼冤枉路。

## 活動內容的黃金骨架

```
1. 破冰與站位：我是誰，為什麼今天這題值得你聽
2. 問題放大：你現在卡住不是你笨，是很多人都卡這裡
3. 錯誤路徑：多數人怎麼做，為什麼沒用
4. 正確框架：真正要看的 3 個關鍵
5. 案例或現場拆解：讓聽眾看到畫面
6. 下一步 CTA：今天結束後，你要去哪裡
```

## 開場原則

好開場不是自我介紹履歷表。
是三分鐘內讓人點頭。

可以用：
- 一個現場故事
- 一句常見誤解
- 一個大家都遇過的卡點

## 中段原則

### 1. 先拆錯，再講對
這樣比較容易讓人感覺你不是在背稿。

### 2. 每 10 分鐘要有一次重新聚焦
- 所以真正要看的不是工具，而是順序
- 所以你卡住的不是內容量，而是判斷力

### 3. 要讓人看到「做完後的生活」
不是只有方法，還要有完成後的畫面。

## 結尾 CTA 原則

結尾不要突然從教學跳成直銷。
最順的收法是：
- 你現在如果還在這個階段，下一步是什麼
- 我已經把那一步整理好了
- 你要的話，現在就往這裡走

## 活動後追單

### 24 小時內
- 回放 / 重點整理
- 常見疑慮回覆
- 明確下一步連結

### 48-72 小時內
- 補一個案例
- 補一個不行動的成本
- 補最後提醒

## 輸出格式

### 格式 A：活動頁文案
- 標題
- 參加對象
- 三個你會帶走的重點
- 報名 CTA

### 格式 B：60 分鐘說明會大綱
- 每 10 分鐘主題
- 開場話術
- 中段轉折
- 結尾 CTA

### 格式 C：直播腳本
- 開場 3 分鐘
- 中段 3 個橋段
- 互動提問
- 收單收尾

### 格式 D：活動後 3 則追擊訊息
- 第 1 則：重點回放
- 第 2 則：案例補強
- 第 3 則：最後 CTA

## 禁忌

- 不要一開場就講自己多厲害
- 不要把活動做成知識 dump
- 不要沒有下一步承接
- 不要把 CTA 分成五條路，聽眾會直接死機
- 不要讓報名與付款流程太繞
