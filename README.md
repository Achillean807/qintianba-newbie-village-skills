# qintianba-newbie-village-skills

秦天霸新手村專用技能組。

這是一套針對台灣市場與個人品牌實戰整理的 Hermes 技能包，主軸不是炫技，而是把內容引流、私域養成、活動轉單、需求挖掘串成一條能真的用的路。

## 這包在做什麼

- 用 Facebook 個人 IP 內容把陌生流量拉進來
- 用 LINE 做名單留存、分眾、後續養成
- 用說明會、工作坊、分享會把興趣推進成行動
- 把複雜主題翻成新手願意看懂的版本
- 用顧問式對話挖需求，而不是一開場就報價硬賣

## 目前包含的技能

1. `台灣FB個人IP引流`
2. `LINE私域成交與分眾養成`
3. `說明會與工作坊轉單`
4. `複雜主題白話轉譯與新手課程包裝`
5. `顧問式需求挖掘與高價服務對話`

## 適合誰

- 在台灣市場做個人品牌的人
- 需要用 FB / LINE / 活動去承接服務或課程的人
- 想把內容、私訊、活動、成交接成同一路線的人
- 想把複雜服務講成人話的人

## 不適合誰

- 想拿去直接跑英文市場的人
- 想要純技術技能庫、完全不碰內容與銷售流程的人
- 想一鍵萬用、不想依自己業務情境調整的人

## 建議用法

### 路線 A：內容引流 → 私訊成交
1. 先用 `台灣FB個人IP引流` 產貼文
2. 用 `複雜主題白話轉譯與新手課程包裝` 降低理解門檻
3. 用 `顧問式需求挖掘與高價服務對話` 承接詢問與成交前對話

### 路線 B：私域養熟 → 活動轉單
1. 用 `LINE私域成交與分眾養成` 留名單與分眾
2. 用 `說明會與工作坊轉單` 做活動頁、流程與收尾 CTA
3. 活動後再用 `顧問式需求挖掘與高價服務對話` 做一對一推進

## 安裝

先把此 repo 放到本機，然後把技能資料夾移到 Hermes 可讀取的位置。

範例：

```bash
mkdir -p ~/.hermes/skills/qintianba
cp -R 01-個人IP引流/台灣FB個人IP引流 ~/.hermes/skills/qintianba/
cp -R 02-私域成交/LINE私域成交與分眾養成 ~/.hermes/skills/qintianba/
cp -R 03-活動轉單/說明會與工作坊轉單 ~/.hermes/skills/qintianba/
cp -R 04-內容轉譯/複雜主題白話轉譯與新手課程包裝 ~/.hermes/skills/qintianba/
cp -R 05-顧問式銷售/顧問式需求挖掘與高價服務對話 ~/.hermes/skills/qintianba/
```
