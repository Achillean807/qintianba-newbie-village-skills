# qintianba-newbie-village-skills

把內容引流、私域養成、活動轉單、顧問式成交，接成一條真的能跑的路。

這不是那種看起來很會、實際落不了地的技能包。
這一包是針對 **台灣市場、個人品牌、FB／LINE／活動承接、服務型成交** 整理的 Hermes 技能組。

如果你平常卡在這幾種場景，這包就是拿來救火的：

- 貼文有流量，卻接不到詢問
- 問的人不少，但最後都只是聊聊
- LINE 加了一堆人，後面不知道怎麼養
- 活動做了，卻沒有把人推到下一步
- 服務很專業，但外行根本聽不懂你在賣什麼

---

## 這包到底在解什麼

它不是單點技能庫。
它比較像一條漏斗骨架，讓你從「有人看到你」一路走到「有人願意成交」。

核心任務只有五件事：

1. 用 Facebook 個人 IP 內容把陌生流量拉進來
2. 用 LINE 做名單留存、分眾、後續養成
3. 用說明會、工作坊、分享會把興趣推進成行動
4. 把複雜主題翻成新手願意看懂的版本
5. 用顧問式對話挖需求，而不是一開場就硬報價

---

## 目前包含的 5 個技能

### 1) 台灣FB個人IP引流
適合：
- 觀點文
- 故事文
- 暖身貼文
- 社團內容
- 導私訊貼文

解的問題：
- 貼文太像廣告
- 沒有人味
- 有內容但沒有引導下一步

### 2) LINE私域成交與分眾養成
適合：
- LINE 官方帳號
- Rich Menu 規劃
- 新好友歡迎流程
- 關鍵字回覆
- 分眾推播

解的問題：
- 名單全混在一起
- 訊息亂發
- LINE 加了人卻沒有後續路線

### 3) 說明會與工作坊轉單
適合：
- 體驗會
- 分享會
- 說明會
- 工作坊
- 活動頁與活動後追擊

解的問題：
- 活動講很多，但收不到下一步
- 開場無力
- CTA 太散，聽眾不知道要幹嘛

### 4) 複雜主題白話轉譯與新手課程包裝
適合：
- AI / 技術服務介紹
- 課程頁
- 說明頁
- 活動介紹
- 專業服務白話版包裝

解的問題：
- 內容太硬
- 專業太多，外行直接關掉
- 聽起來很厲害，但不知道跟自己有什麼關係

### 5) 顧問式需求挖掘與高價服務對話
適合：
- 客戶私訊回覆
- 報價前訪談
- 提案前需求盤點
- 高單價服務銷售對話

解的問題：
- 對方一來就問價格
- 你還沒搞懂需求就開始報方案
- 問半天還是抓不到客戶真正要解什麼

## 你可以怎麼用

### 路線 A：內容引流 → 私訊成交
最適合：個人品牌、顧問、服務型工作者

1. 先用 `台灣FB個人IP引流` 寫內容
2. 再用 `複雜主題白話轉譯與新手課程包裝` 降低理解門檻
3. 最後用 `顧問式需求挖掘與高價服務對話` 承接詢問與成交前對話

### 路線 B：私域養熟 → 活動轉單
最適合：課程、工作坊、說明會、體驗型服務

1. 先用 `LINE私域成交與分眾養成` 做留名單與分眾
2. 再用 `說明會與工作坊轉單` 設計活動頁、流程與收尾 CTA
3. 活動後回到 `顧問式需求挖掘與高價服務對話` 做一對一推進

### 路線 C：專業太硬 → 白話包裝後再引流
最適合：AI、技術、顧問、複雜型服務

1. 先用 `複雜主題白話轉譯與新手課程包裝` 把東西講成人話
2. 再用 `台灣FB個人IP引流` 把內容發出去
3. 問的人進來後，再用 `顧問式需求挖掘與高價服務對話` 往下接

## Quick Start

如果你想先用最短路徑測試，照這個順序就好：

### 內容型業務
- 先裝：`台灣FB個人IP引流`
- 第二個：`顧問式需求挖掘與高價服務對話`

### 活動型業務
- 先裝：`LINE私域成交與分眾養成`
- 第二個：`說明會與工作坊轉單`

### 技術／顧問型業務
- 先裝：`複雜主題白話轉譯與新手課程包裝`
- 第二個：`顧問式需求挖掘與高價服務對話`

如果你懶得選，直接先拿這三個：
- `台灣FB個人IP引流`
- `複雜主題白話轉譯與新手課程包裝`
- `顧問式需求挖掘與高價服務對話`

這組最泛用。

## 安裝方式

### 方式 A：手動安裝到 Hermes

把技能資料夾複製到 `~/.hermes/skills/qintianba/` 底下：

```bash
mkdir -p ~/.hermes/skills/qintianba
cp -R 01-個人IP引流/台灣FB個人IP引流 ~/.hermes/skills/qintianba/
cp -R 02-私域成交/LINE私域成交與分眾養成 ~/.hermes/skills/qintianba/
cp -R 03-活動轉單/說明會與工作坊轉單 ~/.hermes/skills/qintianba/
cp -R 04-內容轉譯/複雜主題白話轉譯與新手課程包裝 ~/.hermes/skills/qintianba/
cp -R 05-顧問式銷售/顧問式需求挖掘與高價服務對話 ~/.hermes/skills/qintianba/
```

### 方式 B：直接下載 Release ZIP

如果你不想用 git，直接去 Releases 下載 zip：

- [Download latest release](../../releases/latest)

解壓後，把需要的技能資料夾搬到 Hermes 的技能目錄即可。

## 適合誰

- 在台灣市場做個人品牌的人
- 需要用 FB / LINE / 活動承接服務或課程的人
- 想把內容、私訊、活動、成交接成同一路線的人
- 想把複雜服務講成人話的人
- 想把銷售從亂槍打鳥，改成比較像顧問式推進的人

## 不適合誰

- 想直接拿去跑英文市場的人
- 想找的是純技術技能庫、完全不碰內容與成交流程的人
- 想一鍵萬用，不想依自己業務情境調整的人
- 想靠花招硬賣，而不是靠內容、結構、對話推進的人

## 建議使用心法

這包最好不要孤立看一顆技能。
真正好用的方式，是把它當成一條連續動線：

- 先讓人願意停下來
- 再讓人聽得懂你在做什麼
- 再讓人留下來
- 再讓人願意往下一步走
- 最後才是成交

如果順序反了，通常就會變成：
內容很努力
活動也辦了
LINE 也加了
結果還是沒單

合理？
